# Thymeleaf Implementations

Thymeleaf is a modern server-side Java template engine for web and standalone environments. It allows developers to create dynamic web applications using plain HTML.

## Implementations and Branches

1. **Basic Thymeleaf Integration**  
   Integrate Thymeleaf with Spring Boot to create web applications easily.

2. **CSS Styling**  
   Add stylesheets to your Thymeleaf templates to enhance the UI.

3. **Form Data Handling**  
   Manage and process form submissions effectively.

4. **MVC Model Data Binding**  
   Bind data to your models and handle it in the view.

5. **Text Input Data Binding**  
   Bind text inputs from forms to your models.

6. **Dropdown Selection Data Binding**  
   Handle dropdown selections in your Thymeleaf forms.

7. **Radio Button Data Binding**  
   Bind radio button selections to your models.

8. **Checkbox Data Binding**  
   Manage checkbox inputs and their data bindings.

## Branch Structure

- **add-css**  
  - Branch for integrating CSS styling into the project.

- **add-form-data**  
  - Branch dedicated to form data handling implementations.

- **add-data-to-mvc-model**  
  - Branch for MVC model data binding functionalities.

- **form-textinputs-data-binding**  
  - Branch focused on text input data binding.

- **data-binding-withother-form-elements**  
  - Branch for handling other form elements like dropdowns, radio buttons, and checkboxes.

For detailed information about each implementation, refer to the respective branches in the repository.
